# OjaReplay Roadmap
**Real-World Commerce Interaction Repository**

OjaReplay is the critical feedback loop of Oja AI Research. It is an anonymized, curated repository of real-world interactions generated by users interacting with MamaPrice. By securely logging user intents, agent tool calls, and final outcomes, OjaReplay continuously feeds high-quality empirical data back into the ecosystem without contaminating the benchmark.

## Current Focus: v0.1 Architecture
- [ ] Define the telemetry and logging schema for agent interactions.
- [ ] Establish strict PII scrubbing and anonymization pipelines.
- [ ] Create a curator interface to flag high-quality interactions.

## Future Milestones
### v0.5: The Replay Pipeline
- Automatically filtering flagged interactions to generate new instruction tuning data for **OjaData**.
- Identifying novel reasoning failures to inspire new, handcrafted benchmark tasks for **ACIB**.

### v1.0: Continuous Improvement Loop
- Creating a fully automated data flywheel where every 10,000 real-world interactions passively improves the next generation of the OjaLM model family.
