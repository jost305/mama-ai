# Oja AI Research Changelog

All notable changes to the OjaLM model, OjaData dataset, and OjaBench evaluation engine will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- **ACIB Schema v1.0**: Formal JSON Schema for African Commerce Intelligence Benchmark supporting multiple evaluation tracks, consumer contexts, and dynamic tool expectations.
- **OjaBench Engine v0.1.0**: Python-based evaluation suite capable of scoring models on factual accuracy, reasoning, completeness, and safety using ACIB v1.0.
- **Commerce Taxonomy v1.0**: Formal mapping of the African commerce sectors and cognitive verticals.
- **Model Requirements Specification (MRS)**: Defined non-functional requirements and deployment boundaries for OjaLM.
