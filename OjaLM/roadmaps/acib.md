# ACIB Roadmap
**African Commerce Intelligence Benchmark**

ACIB is an independent, version-controlled evaluation ecosystem designed to measure commerce intelligence in African markets. It strictly measures reasoning, factual accuracy, and safety across multiple languages, decoupled from the underlying base model.

## Current Focus: v0.1 Research Release
- [x] OjaBench Evaluation Engine
- [x] JSON Schema & Data Governance
- [x] Baseline Infrastructure Separation
- [ ] 100 Food & Groceries Tasks (Food Suite v0.1)
- [ ] ACIB Technical Report Publication

## Future Milestones
### v0.2: The Electronics & Fashion Expansion
- 100 Electronics Tasks
- 100 Fashion & Apparel Tasks

### v0.5: Agentic Tool Use (ACIB-Agent)
- Introducing tasks that require the model to output structured tool calls (e.g., querying a live database) rather than just text.

### v1.0: Multimodal Commerce (ACIB-Multimodal)
- Introducing image-based tasks (e.g., identifying fake products, reading local market packaging).
